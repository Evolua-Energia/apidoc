Envio de Imagem

Nesta requisição é enviado como base o id do lead a ser utilizado, a extensão do arquivo utilizado e base 64 do arquivo destinado.
O seu intuito é que a partir da foto da fatura seja criado sua unidade consumidora com as informações contidas no arquivo.

Requisição Base:

{
	"IdLead" : "1",
	"Extensao: ".extensao",
	"ImagemBase64" : "J465asdkwqb/..."
}

Caminho Negativo Envio Imagem

Sumário

Body Vazio: Algumas das informações ou todas não sejam preenchidas:

{
	"code" : 403,
	"mensagem" : "Conteúdo em branco."
}

Extensão não permitida: É enviado um arquivo diferente das extensões permitidas (pdf, jpeg, jpg, png) :

{
	"code" : 403,
	"mensagem" : "Extensão incorreta."
}

Lead incorreto: O Id do Lead informado não existe ou está incorreto:

{
	"code" : 403,
	"mensagem" : "Lead não existe."
}

Lead não apto à inserção de nova UC: O Lead existe no sistema, porém seu status é igual ao Qualificado ou este Lead já foi convertido:

{
	"code" : 400,
	"mensagem" : "Lead já convertido."
}

Erro ao realizar leitura: Casos possíveis (Mais comum ao menos comum):

  • A qualidade da fatura está muito baixa para extração de dados
  • A imagem inserida é uma base 64 que não é uma fatura
  • Nosso sistema não consegue realizar a extração de dados desta fatura
  • Há um problema na Most para leitura desta fatura, sendo necessário entrar em contato
{
	"code" : 400,
	"mensagem" : "Erro ao realizar leitura."
}

Número de instalação já inserido: Quando uma Unidade Consumidora diferente deste lead já possui este número de instalação:

{
	"code" : 403,
	"mensagem" : "Fatura inválida - Número de instalação já existe."
}

Sub-Classe:Quando a sub classe não é lida na fatura, e, ao buscar um produto, o sistema não identifica um produto que atenda a todas sub-classes, necessitando prosseguir para os campos complementares:

{
	"code" : 403,
	"mensagem" : "Sub classe não legível, prosseguir para campos complementares."
}

Produto: Quando todos os campos obrigatórios para a busca do produto estão preenchidos, porém, ao realizar a busca, não é encontrado um produto ativo para este perfil no momento:

{
	"code" : 403,
	"mensagem" : "Produto não existe."
}

Consumos: Os consumos não foram lidos corretamente (Não foi atingido o critério mínimo para cálculos internos), necessitando prosseguir para os campos complementares:

{
	"code" : 403,
	"mensagem" : "Consumos não conseguiram ser lidos, prosseguir para campos complementares."
}

Ilegibilidade: Os campos mínimos para leitura não foram lidos, nos campos que possuem as variáveis ‘valor’, ‘reenviar’ e ‘pontuacao’, eles irão ser preenchido com os valores contidos no campo dentro do sistema, e caso ele não seja lido o reenviar irá retornar ‘true’ e a pontuação da leitura, neste para fins de exemplo, o número de instalação não foi lido e os outros campos obrigatórios foram lidos:

{
	"unidadeConsumidora" : "UC-XXXXX",
	"status" : "{Status do Lead}",
	"referenciaValorContribuicao" : "0.0xx...",
	"percentualDireitosEconomicos" : "0.0xx...",
	"mensagem" : "Campos mínimos não lidos.",
	"valorEstimado" : "120.0xx...",
	"numeroInstalacao" : {
		"valor" : null,
		"reenviar" : true,
		"pontuacao" : 0.2
	},
	"subClasse" : {
		"valor" : "Residencial",
		"reenviar" : false,
		"pontuacao" : 1.0
	},
	"classe" : {
		"valor" : "Monofasico",
		"reenviar" : false,
		"pontuacao" : 0.98
	},
	"mediaConsumo" : {
		"valor" : "120.0",
		"reenviar" : false,
		"pontuacao" : 1.0
	}
	
}

Inelegibilidade: A fatura não passa pelos critérios de elegibilidade estipulados pela Evolua, necessitando de um processo interno para qualificar o cliente e se tornar elegível e assim seguir com a contratação:

{
	"code" : 403,
	"mensagem" : "Inelegível."
}

Campos Complementares

Nesta requisição é enviado como base o Id do Lead a ser modificado como obrigatório e como opcionais ‘Média de consumo’, ‘Número de instalação’, ‘Classe’ e ‘Sub-Classe’, onde, eles se tornam obrigatórios caso o ‘reenviar’ da requisição de envio de imagem seja ‘true’.

Requisição Base:

{
	"idLead":  "xxxxxxxxx",
	"mediaDeConsumo":  "100.0",
	"numeroInstalacao":  "1234567",
	"classe":  "Monofásico",
	"classificacao":  "Residencial"
}

Resposta Sucesso Campos Complementares

Sumário

{
	"unidadeConsumidora":  "UC-xxxxx",
	"status":  "Email",
	"referenciaValorContribuicao":  "1.0xxxxxx...",
	"percentualDireitosEconomicos":  "1.0xxxxxx...",
	"valorEstimado":  "1.0xxxxxx...",
	"economiaMensal":  "1.0xxxxxx...",
	"economiaAnual":  "1.0xxxxxx..."
}

Caminho Negativo Campos Complementares

Sumário

Body Vazio: O Id do Lead não foi preenchido:

{
	"code" : 403,
	"mensagem" : "Conteúdo em branco."
}

Lead não encontrado: O Lead informado não foi encontrado ou o lead informado não possuí uma unidade consumidora ativa para continuidade da contratação:

{
	"code" : 403,
	"mensagem" : "Lead ou Unidade Consumidora ainda não inseridos no sistema."
}

Lead não encontrado: O Lead informado não foi encontrado ou o lead informado não possuí uma unidade consumidora ativa para continuidade da contratação:

{
	"code" : 403,
	"mensagem" : "Lead ou Unidade Consumidora ainda não inseridos no sistema."
}

Erro em preenchimento : Um campo ilegível que deveria ser informado, não foi informado no ‘Body’ da API:

{
	"code" : 403,
	"mensagem" : "{mensagem_erro}"
}

mensagem_erro:

  • Instalação nula e não preenchida;
  • Classe nula e não preenchida;
  • Sub-Classe (Classificação) nula e não preenchida;
  • Média de consumo nula e não preenchida

Número de instalação já inserido: Quando uma Unidade Consumidora diferente deste lead já possui este número de instalação:

{
	"code" : 403,
	"mensagem" : "Fatura inválida - Número de instalação já existe."
}

Produto: Quando todos os campos obrigatórios para a busca do produto estão preenchidos, porém, ao realizar a busca, não é encontrado um produto ativo para este perfil no momento:

{
	"code" : 403,
	"mensagem" : "Produto não existe."
}

Inelegibilidade: A fatura não passa pelos critérios de elegibilidade estipulados pela Evolua, necessitando de um processo interno para qualificar o cliente e se tornar elegível e assim seguir com a contratação:

{
	"code" : 403,
	"mensagem" : "Inelegível."
}

Simulação: Erro ao realizar a simulação da economia da fatura do cliente, onde ao ocorrer o erro, será necessário análise interna do time Evolua :

{
	"code" : 403,
	"mensagem" : "Erro ao realizar a simulação do plano"
}
